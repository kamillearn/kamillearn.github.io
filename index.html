03. Some Things I've Built

Capstone Project

Education Excellence – Comparative Study of Germany’s Education System

A data-driven analysis of Germany’s 16 federal states to evaluate the quality of education from 2010 to 2022. The study examined correlations between school investments, teacher ratios, and graduation rates using public datasets.

Results showed that increased investment and smaller class sizes generally lead to stronger academic outcomes, while the overall quality of education in Germany has steadily improved.

Python Excel Power BI Data Visualization Statistical Analysis

Data Project

Happy Muesli — EDA & KPI

We analyzed a muesli distributor’s order-to-delivery process and built simple KPIs to track performance. Using historic orders (with partial timestamps) we rebuilt each step — Order → Ready to Ship → On Truck → Delivered — including real-world rules like weekends, truck days (Mon/Wed/Fri) and Express processing.

Python Pandas Jupyter Excel/CSV Data Visualization

Data Project

US Blizzard ’19 – Flights and Weather Data Analysis

We analyzed how extreme weather in North America during the March 2019 blizzard affected flight delays and cancellations. Using flight performance and weather data, our analysis revealed how snow, wind, and temperature variations impacted air traffic across different cities.

The study highlighted key weather factors behind major disruptions and demonstrated how data analytics can help predict and manage such events in the future.

Python Pandas NumPy Matplotlib Data Visualization Jupyter

Data Project

COVID-19 Data Exploration

We analyzed global COVID-19 data using SQL in Google BigQuery to uncover trends in cases, deaths, and vaccination rates. The project explored how the pandemic evolved across countries and how vaccine rollout correlated with reduced infection rates.

The goal was to apply SQL data analysis on large, real-world datasets and present insights through clear summaries and visuals.
